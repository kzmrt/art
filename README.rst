
=======================
Art Works
=======================

This is a Greenhouse Monitor project.


Requirement
===========

:Python: 3.7.0
:Django: 2.1.7
:django-bootstrap4: 0.0.7


Quick start
===========
1. clone project::

    git clone https://github.com/kzmrt/art

2. migrate model::

    python manage.py migrate

3. start server::

    python manage.py runserver
