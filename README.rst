:django-superform: 0.3.1
:Pillow: 5.4.1
:pytz: 2018.9

Quick start
===========
1. clone project::

    git clone https://github.com/kzmrt/art

2. migrate model::

    python manage.py migrate

3. create superuser::

    python manage.py createsuperuser

4. start server::

    python manage.py runserver
